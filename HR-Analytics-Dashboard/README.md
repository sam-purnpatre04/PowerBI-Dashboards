# 📊 HR Analytics Dashboard

## 📌 Project Overview

This project presents an interactive **HR Analytics Dashboard** built using **Microsoft Power BI** to analyze employee attrition and workforce trends.

The dashboard provides HR teams with valuable insights into employee turnover by department, age group, education field, salary slab, and job satisfaction, enabling data-driven workforce decisions.

---

## 🎯 Problem Statement

As an **HR Analyst**, the objective is to study employee attrition and workforce trends across various departments, age groups, education fields, and salary ranges.

The dashboard helps visualize key HR metrics, monitor attrition patterns, and identify factors affecting employee retention.

This project demonstrates skills in:

- Data Preparation

- Interactive Dashboard Design

## 📊 DAX Measures Created

The following KPIs were created using DAX:

- Total Employees
- Employees Left
- Attrition Rate
- Average Monthly Income

These measures provide quick insights into workforce performance.

The dashboard includes:

- 🍩 Donut Chart — Attrition by Education Field
- 📊 Clustered Column Chart — Attrition by Age Group

---

## 💡 Key Business Insights

Some major insights obtained from the dashboard include:

- Sales department recorded the highest employee attrition.
- Employees earning lower salaries showed higher attrition rates.
- Employees aged **26–35 years** experienced the highest turnover.
- Life Sciences had the largest employee population.
- Job satisfaction varied significantly across different job roles.
