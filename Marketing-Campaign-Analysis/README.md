# Marketing Campaign Analysis Dashboard

## Project Overview

This Power BI dashboard analyzes the performance of multiple marketing campaigns across different regions and campaign types.

The dashboard helps businesses evaluate campaign effectiveness using KPIs such as Revenue, Spend, and Return on Investment (ROI).

---

## Tools Used

- Power BI
- Power Query
- DAX
- CSV Dataset

---

## Dataset

The project uses three datasets:

- Marketing_Campaign_Details.csv
- Marketing_Campaign_Performance.csv
- Region_Performance.csv

---

## Dashboard Features

- Total Revenue KPI
- Total Spend KPI
- Average ROI
- Best Performing Campaign
- Revenue vs Spend by Region
- ROI by Campaign
- Campaign Type Distribution
- ROI Trend Analysis
- Interactive Region Filter

---

## Business Insights

- Influencer Marketing generated the highest ROI.
- Africa produced the highest total revenue.
- South America recorded the lowest campaign performance.
- Digital campaigns outperformed traditional campaigns in overall ROI.
